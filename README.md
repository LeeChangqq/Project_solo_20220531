# 기획의도
  - 쇼핑몰 제작을 하면 기능을 만드는데 많은 공부가 될 거 같아서 제작을 하게 되었습니다.
  - 회원가입 및 로그인을 하면 쇼핑몰을 이용을 할 수 있습니다.
  - 간단하게 상품등록은 상품이름, 이미지, 가격, 설명을 작성하여 등록을 하고 
  - 구매 하고 싶은 상품은 장바구니에 담고 구매를 하고 리뷰를 남기는 간단한 쇼핑몰입니다.
# 벤치마킹
  - 온라인 쇼핑몰
